Paquete describes the relationships within the package using a JSON manifest file.</p><p>Paquete uses one page to host all of the pages of a package which reduces overall page loading time and allows users to navigate easily. Once an individual sub-page is loaded, it is cached in the host page making browsing much quicker. </p><p>Specific pages are denoted individually so bookmarking of an individual page is possible. </p><h1><a id="id3" name="id3"><!--id3--></a>What does Paquete consist of?</h1><p>Paquete arranges the files within the package so there is a table of contents (TOC) on the left-hand side and html pages on the right.
