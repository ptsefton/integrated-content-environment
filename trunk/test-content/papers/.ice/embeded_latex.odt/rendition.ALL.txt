    input document.
    <h3>
      <span class="titlemark">1</span> <a id="x1-10001" name="x1-10001"/>Plain Text
    </h3>
    <p>
      Type your text in free-format; lines can be as long or as
      short as you wish. You can indent or space out your input
      text in any way you like to highlight the structure of your
      manuscript and make it easier to edit. LaTeX fills lines and
      adjusts spacing between words to produce an aesthetically
      pleasing result. 
    </p>
    <p>
      Completely blank lines in the input file break your text into
      paragraphs. To change the font for a single character, word,
      or set of words, enclose the word and the font changing
      command within braces, <span class="cmti-10x-x-109">like
      this</span>. A font changing command not enclosed in braces,
      like the change to <span class="cmbx-10x-x-109">bold
      here,</span> <span class="cmbx-10x-x-109">keeps that change
      in effect until the end of the document or until
      countermanded by</span> <span class="cmbx-10x-x-109">another
      font switch, like this change back to</span> roman. 
      
    </p>
    <p/>
    <h3>
      <span class="titlemark">2</span> <a id="x1-20002" name="x1-20002"/>Displayed Text
    </h3>
    <p>
      Use the &#x201C;quote&#x201D; and &#x201C;quotation&#x201D;
      environments for typesetting quoted material or any other
      text that should be slightly indented and set off from the
      normal text.
    </p>
    <div>
      
      <p class="indent">
        The quote and quotation environments are similar, but use
        different settings for paragraph indentation and spacing. 
        
      </p>
      <p class="indent">
        <span class="cmti-10x-x-109">When in doubt, consult the
        manual.</span>
      </p>
    </div>
    <p>
      So far, I have demonstrated titles, paragraphs, font changes,
      and section headings. Now, I am going to show lists and
      tables.
    </p>
    <ol>
      <li class="enumerate">
        <a id="x1-2002x1" name="x1-2002x1"/>The
        &#x201C;enumerate&#x201D; environment numbers the list
        elements, like this. 
        <p class="noindent">
          Items in a list can contain multiple paragraphs. These
          paragraphs are appropriately spaced and indented
          according to their position in the list.
        </p>
        <ul class="itemize1">
          <li class="itemize">The &#x201C;itemize&#x201D; environment
          sets off list items with &#x201C;bullets&#x201D;, like
          this. Finally, the &#x201C;description&#x201D; environment
          lets you put your own
            <dl class="description">
              <dt class="description">
                <span class="cmbx-10x-x-109">A</span>
              </dt>
              <dd class="description">
                label on each item, like this &#x201C;A&#x201D;.
              </dd>
              <dt class="description">
                <span class="cmbx-10x-x-109">If the label is
                long,</span>
              </dt>
              <dd class="description">
                the first line of the item text will be spaced over
                to the right as needed.
              </dd>
            </dl>
          </li>
          <li class="itemize">Of course, lists can be nested, each
          type up to at least four levels. One type of list can be
          nested within another type.
            <ul class="itemize2">
              <li class="itemize">Nested lists of the same type
              will change style of numbering or
              &#x201C;bullets&#x201D; as needed.
              </li>
            </ul>
          </li>
        </ul>
      </li>
      <li class="enumerate">
        <a id="x1-2004x2" name="x1-2004x2"/>Don&#x2019;t forget
        to close off all list environments with the appropriate
        <span class="verb"><span class="cmtt-10x-x-109">\end{...}</span> command. Indenting
        <span class="verb"><span class="cmtt-10x-x-109">\begin{...}</span>, <span class="verb"><span class="cmtt-10x-x-109">\item</span>, and
        <span class="verb"><span class="cmtt-10x-x-109">\end{...}</span> commands in the input
        document according to their nesting level can help clarify
        the structure.</span></span></span></span>
      </li>
    </ol>

    <h3>
      <span class="titlemark">3</span> <a id="x1-30003" name="x1-30003"/>Mathematical Equations
    </h3>
    <p>
      Simple equations, like <span class="cmmi-10x-x-109">x</span><sup><span class="cmmi-8">y</span></sup> or <span class="cmmi-10x-x-109">x</span><sub><span class="cmmi-8">n</span></sub> = <img alt="&#x221A;a-+--b" class="sqrt" src="./latexexample0x.png"/> can be typeset right in the
      text line by enclosing them in a pair of single dollar sign
      symbols. Don&#x2019;t forget that if you want a real dollar
      sign in your text, like $2000, you have to use the
      <span class="verb"><span class="cmtt-10x-x-109">\$</span>
      command. </span>
    </p>
    <p>
      A more complicated equation should be typeset in <span class="cmti-10x-x-109">displayed math</span> mode, like this:
    </p>
    <center>
      <img alt=" ( &#x2218; --------------------- ) x + 1 z (1 + &#x3C9;i+1 + &#x3B6; - -----y + 1 ) = 1 &#x398; + 1 " class="math-display" src="./latexexample1x.png"/>
    </center>
    <p>
      The &#x201C;equation&#x201D; environment displays your
      equations, and automatically numbers them consecutively
      within your document, like this:
    </p>
