Module overview

In this module you will learn some tips and tricks to assist you while using OpenOffice.org Writer, Word and ICE.

  1. Same filename, different extension.

    The system does not support files with the same filename but different extension in the same directory as they cause problems in the manifest. For example, do not use overview.odt and overview.gif in the same directory.

  2. Restart list numbering from 1.

    Writer – Place your cursor in the list item then click the Restart Numbering icon on the Bullets and Numbering bar.

    Word – To restart your numbered list from 1, right click on the first item then select Restart Numbering.

  3. List bullet or number not displaying.

    Writer – After applying the required bulleted list or numbered list style, it may be necessary to use the Bullets On/Off or Numbering On/Off function on the Formatting bar to display the bullet or number.

  4. Use the custom Styles.

    Writer – Attach paragraph and character styles from the custom Styles menu.

    Word – Attach paragraph and character styles from the custom Styles toolbar.

  5. Use an ndash where appropriate. For example, between page numbers (pp. 16–32).

    From the keyboard, press the <Alt> key + <0150> on the number keypad.

    or

    Writer – From the Menu, select Insert...Special Character...Font...Times New Roman...Subset...General punctuation...Click the endash...Click the OK button.

    Word – From the Menu, select Insert...Symbol...Special Characters tab...Click the endash...Click the Insert button.

  6. Use a non-breaking space between items that need to stay together (e.g. pp. 16 or section 3.4.4).

    From the keyboard, press the <Alt> key + <0160> on the number keypad.

    or

    Writer – From the keyboard, press the <Ctrl> key + the spacebar.

    Word – From the keyboard, press the <Ctrl> + <Shift> keys + the spacebar.

  7. Change the quote style.

    Writer – From the Menu, select Tools...AutoCorrect...Custom Quotes tab. Make your selections then click the OK button.

    Word – From the Menu, select Tools...AutoCorrect Options...AutoFormat As You Type tab. Make your selections then click the OK button.

  8. Display alternate text on pictures in a browser.

    Writer – Right click on the picture then select Picture. From the Options tab, locate the Names section and insert a concise meaningful description in the Name text box.

    Word – Alternate text on pictures in Word are not converting through ICE.

  9. Using PNG file format.

    Resizing of interlaced PNG files is not supported. If an interlaced PNG is used/required, the actual size of the image is displayed in ICE and not the resized version in the document.

  10. Inserting pictures.

    Writer – When inserting a picture, ensure the Link option is not selected. To check if you have any linked pictures in your document, from the Menu, select Edit...Links. (The Links option will only be visible if there are linked pictures in the document.). If you have linked pictures, unlink them.

    If you receive an ICE error which tells you to restart OpenOffice.org, check if you have linked images.

  11. Table spanning problems.

    Writer – If you had made the table width relative at 100%, then set the table alignment to automatic, the table will span 100%. But, when the document is changed, OpenOffice.org Writer sets the table width back to cm if the alignment is set to automatic.

    Therefore, we recommend that if a table width needs to be relative (as a percentage), don't set the alignment to automatic (e.g. select Left instead of Automatic).

  12. Document property changes and header update.

    The ICE templates left and right headers have been set up with fields. The fields include the information you insert in the Title and Subject properties. From the Menu, select File...Properties.

    Writer – Changes to the property fields are automatically reflected in the headers after you OK out of the dialog box.

    Word – Document headers may not show the changes made to the properties until the view is refreshed. A quick way is from the Menu, select View...Web Layout then View...Print Layout. You should now see the revised data.

  13. Vertical table cell merge.

    Avoid vertical table cell merges in Word and Writer. OpenOffice.org does not handle this correctly in the conversion to web. Horizontal merging works correctly.

  14. Using GIF file format on a Mac.

    Resized GIFs on the Mac are not supported in ICE. You could try saving/exporting your problem images to JPG format.

  15. Activation of X11 – Mac.

    Activate X11 prior to opening OpenOffice.org. An white icon with a X will be on the taskbar at the bottom of the screen. This will activate the Alt key which you can use for menu keyboard shortcuts.

  16. Word list numbering not correct in Writer master document.

    We are aware of the problem and the development team are working to resolve this issue. Open the Word document in Word and run the Repair Lists macro (access from the Macros option on the ICE toolbar). Update the links in the master document. If there is still a problem with lists not performing correctly there is a temporary manual fix.

    Open the Word document in Word. Create a dummy list prior to the first real list. Highlight each complete list separately in the document then right click and select Restart Numbering. Remove the dummy list. Save and close the document then update links in the Writer master document to view the changes.

  17. Turn off the display of field codes.

    The headers use Fields that relate to the document properties (Title and Subject). If the field code display is turned on, you will not see the text you included in the properties but the field code only. Use the following process to turn off the display of field codes.

    Writer – From the Menu, select Tools...Options. From the left hand pane, select OpenOffice.org Writer (expand if necessary) then select View. From the right hand pane Display section, deselect the Field codes option then click the OK button.

    Word – From the Menu, select Tools...Options. Select the View tab. From the Show section, deselect the Field codes option then click the OK button.

  18. Empty table cells.

    Include a non-breaking space in empty table cells or they will not convert correctly in the web.

    Writer – From the keyboard, press the <Ctrl> key + the spacebar.

    Word – From the keyboard, press the <Ctrl> + <Shift> keys + the spacebar.
